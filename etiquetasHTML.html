Etiquetas HTML


Al inciar a crear primero tenemos que crear la primera página web, para ello tenemos que iniciar con html, lo primero que tenemos que hacer es ir al editor de código y crear un archivo html.


La página principal de una página web se le pone el nombre de index, por lo que el nuevo archivo deberia quedar así:
index.html


Una vez configurado el documento, ya podemos iniciar a escribir, html utiliza etiquetas (<_>)para que lo que escribamos se muestre al abrir en el documento. Algunas etiquetas tienen que abrirse y cerrarse (<__>), mientras que otra solo se añaden la etiqueta de apertura (<__>).
